# STACK_SELECTION.md

Before creating any application, ALWAYS ask:

1. App Type
- Web
- Mobile
- Fullstack
- Backend API

2. Frontend Stack
- Next.js
- React
- None

3. Backend Stack
- Laravel
- NestJS
- None

4. Mobile Stack
- Flutter
- Kotlin
- None

5. Database
- PostgreSQL local
- NeonDB

Never assume stack automatically.
Always confirm with user first.
