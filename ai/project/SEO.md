# SEO.md

SEO Strategy:
Portfolio website must be optimized for:
- Google Search
- LinkedIn profile visits
- Recruiter discovery
- Personal branding visibility

Primary Keywords:
- Anastasia Liem
- Talent Acquisition Specialist
- Human Resources Professional
- HR Operations
- Recruiter Indonesia
- Talent Acquisition Indonesia
- Human Resource Representative

Secondary Keywords:
- Recruitment Specialist
- Candidate Assessment
- HR Administration
- Talent Sourcing
- Employee Relations

SEO Rules:

Page Title:
Format:
{Name} | {Professional Role}

Example:
Anastasia Liem | Talent Acquisition & HR Operations

Meta Description:
Length:
140–160 characters

Template:
Professional portfolio of Anastasia Liem, experienced in Talent Acquisition, Recruitment Operations, and Human Resources administration.

Heading Rules:
- Only 1 H1 per page
- Section titles use H2
- Subsections use H3

URL Rules:
Use clean URLs.

Good:
/
#/about
#/experience
#/contact

Bad:
/page?id=123

Image Optimization:
- Compress images
- Use modern formats (webp)
- Add alt text

Example Alt Text:
Anastasia Liem professional portrait

Performance Goals:
- Lighthouse score > 90
- LCP < 2.5s
- CLS < 0.1

Schema Markup:
Use structured data:
- Person
- Organization
- Website

Open Graph:
Required for social sharing.

OG Title:
Anastasia Liem Portfolio

OG Description:
Talent Acquisition & Human Resources Professional

SEO Avoid:
- Keyword stuffing
- Duplicate meta descriptions
- Large unoptimized images
